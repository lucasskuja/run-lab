# Contributing

Thanks for contributing to `cloud-run-python-service-template`.

This repository is maintained as a reusable reference implementation for lightweight Python services on Google Cloud Run. Changes should preserve that goal: keep the structure clear, the deployment path simple, and the examples practical to adapt.

## Contribution Guidelines

- Open a descriptive pull request with the technical intent of the change.
- Keep documentation aligned with the current implementation.
- Prefer changes that improve reusability, clarity, or extensibility.
- Avoid adding project-specific assumptions that reduce template value.
- Run tests before submitting a pull request.
- Document behavioral or structural changes that affect how this repository should be reused.
